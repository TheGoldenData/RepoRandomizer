# REPO Randomizer
Random effects that trigger in levels.

## Multiplayer
EVERYONE NEEDS IT to work properly, otherwise players won't expierience effects.
All effects are synchronized, meaning everyone has same effect at the same time.

## Configuration
Configure the mod through the BepInEx config file. It is recommended to use the REPOConfig UI for easier configuration in-game.

**Setting include:**
- Interval between effects
- Each effect duration
